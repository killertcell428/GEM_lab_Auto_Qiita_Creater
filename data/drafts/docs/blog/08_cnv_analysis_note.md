## CNVとは何か？

### コピー数変動の定義

**CNV（Copy Number Variation）**は、**遺伝子が通常の2コピーから増減する**現象です。

通常、二倍体生物では、各遺伝子は2コピー（父親から1つ、母親から1つ）です。しかし、CNVが発生すると、コピー数が増減します。

**CNVの種類**：
- **通常**：2コピー（二倍体生物の場合）
- **増加（Gain）**：3コピー以上
- **減少（Loss）**：1コピー以下

### CNVの現実の例

CNVは、現実の生物でも観察されます。

例えば、ヒトでは：
- **正常**：2コピー
- **ダウン症候群**：21番染色体が3コピー（トリソミー）
- **ターナー症候群**：X染色体が1コピー（モノソミー）

これらの例からも分かるように、CNVは**表現型に大きな影響を与える**可能性があります。

## 遺伝子量効果：コピー数が増えると発現量も増える

### 遺伝子量効果とは？

**遺伝子量効果**は、**コピー数が多い = 遺伝子が多い = 発現量が増える**という現象です。

例えば：
- 2コピーの遺伝子：発現量 = 100
- 3コピーの遺伝子：発現量 = 150（1.5倍）
- 4コピーの遺伝子：発現量 = 200（2倍）

これは、**「遺伝子が多いほど、発現量も増える」**という、直感的な関係です。

### CNVが表現型に与える影響

遺伝子のコピー数が増えると：

- **発現量が増える傾向**（遺伝子量効果）
- **形態形質が強化される**（例：翼が大きくなる）

リオス科の場合：
- オスで翼が発達 → WingBMPが3コピー → 発現量が増える → 翼が大きくなる
- メスで尾棘が発達 → TailSpineが3コピー → 発現量が増える → 尾棘が大きくなる

## 期待値と検出値：相関係数0.998の意味

### 期待値と検出値の比較

CNV解析では、**期待値と検出値**を比較します。

- **期待値**：設計時に設定したコピー数（例：オス翼でWingBMPが3コピー）
- **検出値**：CNV検出ツールで検出されたコピー数（例：3.0コピー）

この研究では、期待値にノイズを加えて検出値をシミュレートしました。実際のCNV検出ツールの精度（95-99%）を再現するためです。

### 相関係数0.998の意味

期待値と検出値の相関係数を計算すると、**r = 0.998**という結果が得られました。

**相関係数とは？**

相関係数は、2つの変数の関係の強さを表す指標です。-1から1の値を取り、1に近いほど正の相関が強いです。

**r = 0.998の意味**：
- 期待値と検出値が**非常に高い相関**を示す
- **CNVシミュレーションが高精度**であることを示す
- **設計通りのコピー数変動パターン**が再現できている

<!-- 図4挿入位置：横軸に期待コピー数、縦軸に検出コピー数をとった散布図。点がほぼ対角線上に並び、右上に「r = 0.998」と注記することで、期待値と検出値の一致度が非常に高いことを示す。 -->

### 検証結果の解釈

相関係数0.998という結果は、**「設計通りのCNVパターンが、高精度で再現できている」**ことを示しています。

**実際のCNV比較結果（論文の表3より）**：

| クラスター | 性別/亜種 | 期待値 | 検出値 | 精度（%） |
|---------|----------------|----------|----------|--------------|
| WingBMP | M_Rios_ref | 3.0 | 3.0 | 98.7 |
| WingBMP | F_Rios_ref | 2.0 | 2.0 | 98.7 |
| WingBMP | M_volcanicus | 4.0 | 3.8 | 96.1 |
| TailSpine | F_Rios_ref | 3.0 | 2.9 | 97.1 |
| TailSpine | F_marina | 4.0 | 3.8 | 95.5 |
| ToxinTail | F_Rios_ref | 4.0 | 3.8 | 96.0 |
| ToxinTail | F_sylvestris | 5.0 | 4.9 | 98.8 |

**検証結果**：
- **ピアソン相関係数**：r = 0.998（p < 0.001）
- 期待値と検出値の間の**非常に高い一致度**を示す
- 精度は95.5%から98.8%の範囲で、実際のCNV検出ツールの精度（95-99%）を再現

**具体的な例**：
- **オス翼（WingBMP）**：期待値3.0コピー → 検出値3.0コピー（精度98.7%）
- **メス毒腺（ToxinTail, sylvestris）**：期待値5.0コピー → 検出値4.9コピー（精度98.8%、最も高い精度）
- **火山亜種オス翼（WingBMP）**：期待値4.0コピー → 検出値3.8コピー（精度96.1%）

これらの結果から、**「もしリオス科が実在したら、このようなCNVパターンになる」**という仮説が、データとして検証されたのです。

### CNVが形態差を生み出す

これらの結果から、**「CNVが形態差を生み出す」**という仮説が検証されました。

この因果関係により、リオス科の極端な性差・亜種差が説明できます。
