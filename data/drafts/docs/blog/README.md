# ブログ記事シリーズ「仮想ゲノム解析で学ぶRNA-seq解析」

## 概要

このディレクトリには、仮想ゲノム解析プロジェクトを題材にしたRNA-seq解析の学習用ブログ記事シリーズが含まれています。

## 記事一覧

1. **[第1回：イントロダクション](01_introduction.md)**
   - プロジェクトの背景と目的
   - 使用する技術スタックの概要
   - プロジェクト構成図

2. **[第2回：環境構築とデータ準備](02_environment_setup.md)**
   - conda環境の構築手順
   - バックボーンゲノムの取得方法
   - 染色体リネームの実装

3. **[第3回：トランスクリプト設計とRNA-seqシミュレーション](03_simulation.md)**
   - 14遺伝子の選定理由と設計
   - トランスクリプトFASTAの作成方法
   - polyesterによるリードシミュレーション

4. **[第4回：アライメントとカウント](04_alignment_counting.md)**
   - トランスクリプトインデックスの作成理由
   - HISAT2アライメントの実行
   - カスタムカウント関数の実装

5. **[第5回：発現解析と統計検定](05_deseq2_analysis.md)**
   - DESeq2のデザイン式の選択
   - モデル行列のフルランク問題と解決策
   - 性差・組織差・交互作用の解釈

6. **[第6回：可視化と解釈](06_visualization.md)**
   - 火山図の読み方と作成方法
   - ヒートマップによる発現パターンの可視化
   - PCAによるサンプル間の関係性

7. **[第7回：CNV解析とスライド用データ作成](07_cnv_analysis.md)**
   - CNV領域の定義（BEDファイル）
   - 期待コピー数の設計（性差・亜種差）
   - 検出値のシミュレーション

8. **[第8回：まとめと今後の展望](08_summary.md)**
   - プロジェクト全体の成果サマリー
   - 学んだ技術と知識の整理
   - 今後の展開
